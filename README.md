### Introduction

This repository houses the code and the assets used to build the Yaremche Farmers Market website.

On the webpage the users can find general information about the farmers market and it's location, the subscription options that are available and more. The users can contact the sponsor via email or by using the contact form as well as to submit a participation request. 

The website was built using HTML, Bootstrap and Formspree forms. And it is a great example of how limited tools can be used to create simple and yet beautiful websites.
